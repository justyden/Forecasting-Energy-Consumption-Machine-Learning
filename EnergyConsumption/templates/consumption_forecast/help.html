Application Overview

This project aims to develop a machine learning-based solution for forecasting energy consumption. Historical energy consumption data will be analyzed to identify patterns and trends using data visualization techniques. Two phases of machine learning algorithms will then be implemented. The first phase algorithm will be used for classification to predict energy consumption categorized as being periods of typical, higher, or lower than normal energy consumption. The second phase algorithm will then be used to quantitatively forecast energy consumption in megawatt hours (MWh).


Key Features

  • SVM (Support Vector Machines): Checks if energy consumption is low or high.
  • Linear Regression: Gives important statistical information.
  • Website Implementation: Developed using Django.
